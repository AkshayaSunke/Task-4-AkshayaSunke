# General Knowledge Quiz

## Project Description
The General Knowledge Quiz is a simple Python application that asks users three general knowledge questions. The program accepts user input, checks the answers, calculates the score, and displays the final result.

## Objective
The objective of this project is to demonstrate the use of basic Python programming concepts, including:
- Variables
- User Input
- String Methods (`strip()` and `lower()`)
- Conditional Statements (`if-else`)
- Score Tracking
- Formatted Output using f-strings

## Features
- Asks three general knowledge questions.
- Accepts user input from the keyboard.
- Validates answers regardless of letter case and extra spaces.
- Calculates and displays the final score.
- Provides feedback based on the user's performance.

## Technologies Used
- Python 3

## How to Run
1. Save the Python code in a file named `quiz.py`.
2. Open a terminal or command prompt.
3. Navigate to the project directory.
4. Run the following command:

```bash
python quiz.py
```

## Sample Questions
1. What is the capital of France?
2. Which planet is known as the Red Planet?
3. What is the largest ocean on Earth?

## Expected Output
The program informs the user whether each answer is correct or incorrect and displays the final score at the end of the quiz.

## Learning Outcomes
After completing this project, learners will be able to:
- Take input from users.
- Use conditional statements for decision-making.
- Process and compare strings.
- Maintain and update variables.
- Build a simple interactive Python application.
